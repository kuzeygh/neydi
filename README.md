## neydi ya?
a command line helper for our absent minded friends

# install
`python3 setup.py install --record tracing.txt`

# uninstall
`python3 uninstall.py`

# dev 
`source neydi/venv/bin/activate`

`python setup.py install --record tracing.txt`

> installs to venv

`neydi --help`
> runs from venv
